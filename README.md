# API 요구기능
- CRUD(Create Read Update Delete)
    - 카드 조회
        - 입력: x
        - 반환: 전체 카드 리스트
    
    - 카드 생성
        - 입력: 카드객체
        - 반환: id 반환
    
    - 카드 업데이트
        - 입력: 카드객체
        - 반환: x

    - 카드 삭제
        - 입력: id
        - 반환: X

# 카드 클래스 정의
- id
    - 카드 id
- category
    - todo / ongoing / test / done
- title
    - 카드 내용
- cardElement
    - HTML/DOM
